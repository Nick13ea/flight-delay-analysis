# Flight Delay Analysis Project

## Overview
This project analyzes airline flight delay data using Python to identify carrier performance trends.

## Tools Used
- Python
- Pandas
- Matplotlib

## Process
- Loaded Parquet dataset
- Cleaned missing values
- Engineered total delay feature
- Aggregated delays by airline carrier
- Visualized results

## Output
- Carrier delay comparison chart in `/visuals`

## Insight
Identified best and worst performing airlines based on average delays.
